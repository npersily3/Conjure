# dev.conjure.compat — third-party mod integrations

Optional integrations with other mods. Each integration lives in its own sub-package and is only
exercised when that mod is present on the classpath.

## Sub-packages

| Package | Purpose |
|---------|---------|
| `jei/` | [JEI](https://www.curseforge.com/minecraft/mc-mods/jei) integration: shows Conjure workbench recipes (and the workbench blocks) in the recipe viewer. |
